# OData 4 Demo with Spring Boot & React

这是一个基于Spring Boot和React的OData 4示例项目，实现了订单的列表展示和数据提交功能。

## 功能特性

- 订单（订单头，订单项）（支持PC和移动端）（详情，列表）
- OData 4.0 标准接口
- TypeScript 支持
- Ant Design Pro Components 集成
